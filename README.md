# 一个基于react native 的dota视频app

mac:

### Install


```
npm install

react-native run-ios

```

**windows 下目前无法运行。**


#### 目标：

基于react native 开发一款兼容ios和android的dota视频app。

#### 已完成（仅ios）：
视频列表、主播列表、主播详细、视频播放
